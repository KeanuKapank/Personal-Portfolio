# 🧠 Personal Portfolio Project
This project is a personal portfolio application built with React, utilizing client-side routing with React Router DOM, and styled with Tailwind CSS. It is designed to be flexible, responsive, and easy to navigate, providing a clean and modern user interface.


## 🛠️ Tech Stack
* Frontend: React, React Router DOM, Tailwind CSS
* Build Tool: Vite

## 📦 Installation
To get started with the project, follow these steps:
1. Clone the repository to your local machine
2. Run `npm install` to install the dependencies
3. Run `npm run dev` to start the development server
